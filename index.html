Rules

You have 50 seconds to enter an expression that gives the target number (or closest possible one) by using 6 given numbers and basic algebraic operations (+, -, *, /), including parenthesis. You don't need to use all the numbers.

The game ends when the Finish button is clicked, or when the time is up. When the game is over, a solution generated by the computer will be displayed.

Fractions and negative numbers are not allowed.

Only games with random numbers are included in the stats. The solution is considered correct if the expression gives the target number (or the closest possible one).
